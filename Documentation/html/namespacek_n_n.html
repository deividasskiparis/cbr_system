k-nearest neighbour (kNN) function.

Compares a new case in the feature space and returns k most closely matching instances, based on the selected similarity measure

Parameters
case_basenumpy array of data of shape (row c column) N x D, where N is instances and D - dimensionality
new_caseA new instance of data to be compared to all instances in case_base [1 x D]
kNumber of nearest neighbours
dist_measSimilarity measure. Avalable values are "EUCL+W", "EUCL", "MANH+W", "MANH", "LEIX" for Euclidean Manhattan and Eixample similarity measure. +W = Weighted.
attr_weightsattribute weights for weighted retrieval. If the dist_meas has "+W" suffix or is "LEIX", this is required.
EIX_mapAn array, containing information about data. Created using make_Eixample_map.py
eix_alphaEixample threshold. Numerical features with weights above this threshold are treated as lineal
Return values
distsDistances from k most similar cases
idxsRow indices of the most ismilar cases in the case_base
new_case_allNumAll numerical representation of new_case. Used for Eixample distance measure.
case_base_allNumAll numerical representation of retrieved cases. Used for Eixample distance measure.
